개인정보처리방침
시행일: 2025년 1월 1일

1. 수집하는 정보

치지직 룰렛(이하 "서비스")은 다음 정보를 수집합니다.

  • 치지직 채널 ID (사용자가 직접 입력)
  • 공개 채널명 (치지직 공개 API를 통해 조회)
  • 브라우저 로컬 스토리지에 저장되는 마지막 조회 채널 ID

2. 정보 이용 목적

  • 채널 정보 조회 및 룰렛 기능 제공
  • 최근 조회 채널 자동완성 편의 기능 제공

3. 정보 보관

채널 ID는 사용자의 브라우저 로컬 스토리지에만 저장되며, 서버에 전송·저장되지 않습니다. 언제든지 브라우저 설정에서 삭제할 수 있습니다.

4. 제3자 제공

수집된 정보는 제3자에게 제공되지 않습니다. 단, 채널 조회 시 치지직(NAVER) 공개 API가 사용됩니다.

5. 실시간 채팅 데이터

실시간 채팅 및 후원 데이터는 화면 표시 용도로만 사용되며, 서버에 저장되지 않습니다.

6. 쿠키 및 추적

본 서비스는 쿠키를 사용하지 않으며, 사용자 행동을 추적하지 않습니다.

7. 문의

개인정보 관련 문의는 서비스 운영자에게 연락해 주세요.
