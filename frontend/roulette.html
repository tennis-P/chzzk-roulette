개인정보처리방침
시행일: 2025년 1월 1일

1. 수집하는 정보

치지직 룰렛(이하 "서비스")은 다음 정보를 수집합니다.

  • 치지직 채널 ID (사용자가 직접 입력)
  • 공개 채널명 (치지직 공개 API를 통해 조회)
  • 브라우저 로컬 스토리지에 저장되는 마지막 조회 채널 ID

2. 정보 이용 목적

  • 채널 정보 조회 및 룰렛 기능 제공
  • 최근 조회 채널 자동완성 편의 기능 제공

3. 룰렛 항목 추가 방식

룰렛 항목은 아래 두 가지 방식으로 추가됩니다. 시청자의 닉네임은 수집하지 않습니다.

  • 채팅 명령어: 시청자가 채팅에 명령어 + 항목명을 입력하면, 명령어 뒤의 텍스트(항목명)만 룰렛에 추가됩니다. (예: !룰렛추가 치즈 → "치즈" 추가)
  • 후원 메시지: 시청자가 후원 시 작성한 메시지 내용이 룰렛 항목으로 추가됩니다. 후원자 닉네임은 사용하지 않습니다.

4. 정보 보관

채널 ID는 사용자의 브라우저 로컬 스토리지에만 저장되며, 서버에 전송·저장되지 않습니다. 룰렛 항목은 페이지를 벗어나면 초기화되며 어디에도 저장되지 않습니다.

5. 제3자 제공

수집된 정보는 제3자에게 제공되지 않습니다. 단, 채널 조회 시 치지직(NAVER) 공개 API 및 채팅 WebSocket 서버가 사용됩니다.

6. 쿠키 및 추적

본 서비스는 쿠키를 사용하지 않으며, 사용자 행동을 추적하지 않습니다.
