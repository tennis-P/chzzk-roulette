# 🎰 치지직 룰렛

치지직(Chzzk) 스트리머를 위한 **실시간 시청자 참여형 룰렛 시스템**입니다.  
후원 메시지와 채팅 명령어를 통해 시청자가 직접 룰렛 항목을 추가할 수 있습니다.

---

## ✨ 주요 기능

### 🎡 룰렛
- 항목별 칸 수 조절 (+/- 버튼)
- 각 항목의 당첨 확률(%) 실시간 표시
- 부드러운 스핀 애니메이션
- 당첨 결과 오버레이 팝업

### 💬 채팅 명령어
- 시청자가 채팅에 명령어를 입력하면 룰렛 항목 자동 추가
- 명령어 직접 설정 가능 (기본값: `!룰렛추가`)
- 예시: `!룰렛추가 치즈` → **"치즈"** 항목 추가
- ON/OFF 토글로 켜고 끌 수 있음

### 🎁 후원 연동
- 시청자 후원 시 **메시지 내용**이 룰렛 항목으로 자동 추가
- 칸당 치즈 금액 설정 (예: 100치즈 = 1칸)
- 시청자당 최대 칸 수 제한 설정 가능
- ON/OFF 토글로 켜고 끌 수 있음

### 🖥️ UI / 편의 기능
- 헤더 슬라이더로 화면 배율 조절 (80% ~ 150%)
- 최근 조회한 채널 ID 자동 저장 및 빠른 입장
- 화면 확대/축소 줌 컨트롤

---

## ⚙️ 설정 방법

### 채널 ID 확인
치지직 채널 URL에서 확인할 수 있습니다.  
예) `https://chzzk.naver.com/live/abc123def456...` → `abc123def456...`

### 스트리머 설정 패널 (룰렛 페이지 좌측)

| 항목 | 설명 |
|------|------|
| 자동 추가 | 후원 시 룰렛 칸 자동 추가 ON/OFF |
| 칸당 치즈 | 이 금액당 1칸이 추가됩니다 |
| 최대 칸 수 | 동일 항목의 최대 칸 수 제한 |
| 명령어 사용 | 채팅 명령어 기능 ON/OFF |
| 명령어 설정 | 원하는 명령어로 변경 가능 |

## 🔒 개인정보처리방침

- 채널 ID는 브라우저 로컬 스토리지에만 저장되며 서버로 전송되지 않습니다.
- 룰렛 항목은 페이지를 벗어나면 초기화되며 어디에도 저장되지 않습니다.
- 시청자 닉네임은 수집하지 않으며, 후원 메시지 내용과 채팅 명령어 뒤의 텍스트만 룰렛 항목으로 사용됩니다.
- 자세한 내용은 서비스 내 **개인정보처리방침**을 확인해 주세요.
